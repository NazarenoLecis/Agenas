# Source notes

Questo file contiene note operative sulle fonti.

## AGENAS

AGENAS contiene cruscotti, report e indicatori. Non tutti i contenuti sono dataset scaricabili. Ogni fonte deve essere verificata prima di abilitarne il download automatico.

## Ministero della Salute

Il portale Open Data del Ministero della Salute e una fonte prioritaria per dataset scaricabili. Quando esistono CSV, JSON o XML ufficiali, gli script possono scaricarli e salvarli in data/raw.

## Regioni

Le fonti regionali vanno aggiunte solo dopo avere verificato licenza, struttura del dato, granularita e stabilita degli URL.
